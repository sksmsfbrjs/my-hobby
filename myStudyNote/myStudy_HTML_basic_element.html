스타일(style)

HTML 요소의 style 속성(attribute) 이용 시 css 스타일을 HTML 요소에 직접 설정 가능
(but, 단 하나의 요소에만 사용 가능)

문법

<태그이름 style="속성이름 : 값">

ex> 배경색 변경

<body style="background-color: aqua">
<h1 style="background-color: white">HI</h1>
</body>

ex> 글자색 변경

style="color: pink;"

ex> 글자 크기 변경

style="font-size: 200%;"

ex> 문단정렬 변경

style="text-align: center;"

* 만약 여러스타일을 한번에 하려면
style="background-color: pink; font-size: 150%;"
