제목(Title)

h1~h6까지 있음. h 위아래로 여백 생성됌. 제목의 표현기능 외에 검색엔진의 키워드 수집의 대상이 됌.

단락(Paragraph)

p : 단락을 나눌 땐 br태그로 줄바꿈하기 보다는 p태그를 이용하여 사용하는게 중요함 단락 경계를 분명히 하고 css를 통해 p태그 디자인 변경 가능.
단점은 단락과 단락의 간격이 고정되어 CSS의 margin을 이용해야 함.

br : 줄바꿈 태그, empty tag임.

pre (preformatted text) : HTML코드에서 작성한 서식을 그대로 표현

hr : 수평 가로 구분선

del : 이렇게 뜸

ins : 이렇게 뜸

* 짧은 인용구

주석 표현 (Commet)

주석 : 코드에 대한 설명 or 디버깅을 위해 작성되는 구문
1. HTML에서의 작성법 -> <!-- 주석내용 -->
2. CSS에서의 작성법 -> /* 주석 내용 */

엔티티(Entity)

HTML 예약어를 기존에 사용하던 의미 그대로 사용하기 위해 별도로 만든 문자셋
(HTML 예약어를 HTML코드에서 그대로 사용하면 웹브라우저는 평소와 다르게 해석!)

* 엔티티 형태
-> &엔티티이름; or &#엔티티숫자(16진수);

> https://www.w3.org/TR/html4/sgml/entities.html 이 사이트에 들어가면 entity에 관한 내용이 나옴.

문자셋(Character set)

웹 브라우저가 HTML 문서를 정확하게 나타내기 위해서는 해당 문서가 어떠한 문자셋으로 저장되었는지를 알아야 합니다.
따라서 HTML 문서가 저장될 때 사용된 문자셋에 대한 정보를 <head>태그 내의 <meta>태그에 명시합니다.

→ 문자셋의 종류

현재 사용되는 대표적인 문자셋(character set)은 다음과 같습니다.
1. ASCII : 가장 처음 만들어진 문자셋으로, 인터넷에서 사용할 수 있는 127개의 영문자와 숫자로 이루어져 있습니다.
2. ANSI : 윈도우즈에서 만든 문자셋으로, 총 256개의 문자 코드를 지원합니다.
3. ISO-8859-1 : 256개의 문자 코드를 지원하는 HTML4의 기본 문자셋입니다.
4. UTF-8 : 세상에 있는 거의 모든 문자를 표현할 수 있는 유니코드 문자를 지원하는 HTML5의 기본 문자셋입니다.
