생활코딩 CSS편

CSS는 HTML이 정보의 기능에 전념하도록 디자인의 기능을 전체적으로 담당함. 효율적임!

CSS의 기능

  1. HTML 속성(style) 이용
  2. 태그 <style> 이용

ex> 글꼴 변경 -> style="color:색이름"

만약 모든 링크의 색깔을 변경해야 한다면?

<head>
    <style>               ┐
        a {color:색이름;} │-------> CSS 역할
    </style>              ┘
</head>
a {color:색이름;} 에서 a는 선택자(Selector), color는 속성(Property), 색이름은 속성값(Property Value)이며, a 선택자에 color:색이름 으로 선언(Declaration)한다고 칭한다.

만약 모든 링크의 밑줄을 없애고 싶다면 CSS style 태그에 text-decoration: none; 추가!
(특정 하나만 긋고 싶다면 HTML 코드 내 <a style="text-decoration: underline"> 추가!)

7. CSS 선택자의 기본

<a style="color:red">HTML</a>
<a style="color:red"></a>
이렇게 쓰게 될 경우 코드가 중복됌!

따라서 클래스를 정의 한 후 클래스를 적용!

class: 그룹으로 묶어서 스타일을 지정할 때 쓰는 선택자.
id 선택자: 스타일을 지정할 때 한 가지만 지정해서 쓰는 이름 , 중복 x!

ex> 
<style>
    a {
        ~
    }
    .saw {             -> .class이름 선택자
        color:red;
    }
    .active {
        color:gray;
    }
    #gogo {             -> #id 선택자
        color:blue;
    }

    ...

<a ~ class="saw active" id="gogo">HTML</a>

실행 순서

#id 선택자 > .class이름 선택자 > 태그 선택자

→ 만약 id가 없다면 CSS style 태그에서 실행 순서가 saw가 active보다 앞에 있으므로 나중 순서인 active가 적용됌.
설계 시, 전체적으로 틀을 짠 후 예외적인 것들을 짜는게 효율성이 좋다!
