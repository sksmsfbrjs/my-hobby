제목(Title)

h1~h6까지 있음. h 위아래로 여백 생성됌. 제목의 표현기능 외에 검색엔진의 키워드 수집의 대상이 됌.

단락(Paragraph)

p : 단락을 나눌 땐 br태그로 줄바꿈하기 보다는 p태그를 이용하여 사용하는게 중요함 단락 경계를 분명히 하고 css를 통해 p태그 디자인 변경 가능.
단점은 단락과 단락의 간격이 고정되어 CSS의 margin을 이용해야 함.

br : 줄바꿈 태그, empty tag임.

pre (preformatted text) : HTML코드에서 작성한 서식을 그대로 표현 hr : 수평 가로 구분선

서식(Formatting)

* 강조효과

b : 단순 굵게 표현
strong : 굵게 표현 & 검색엔진 중요도 높음

* 이태리체

i : 단순 표시
em : 이태리체 & 검색엔진 중요도 높음

* 하이라이트 효과

mark : 이렇게 뜸

* 삭제효과

del : 이렇게 뜸

* 삽입효과

ins : 이렇게 뜸

인용구

* 짧은 인용구

q : 앞뒤에 큰따옴표

* 블록 인용구

길이가 긴 인용구는 <blockquote>태그 사용. 별도의 단락으로 처리됌

* 축약형 표현(abbreviation)

abbr : abbr 태그가 적용된 문구에 마우스를 대면 풀네임이 나옴
ex> <abbr title="풀네임">HTML</abbr> => HTML

* 주소 표현

address : HTML에서 주소 표현 가능
ex> <address> 서울시 <br> 강남구 </address>
->
서울시 강남구
(위 아래로 공백이 자동 생성됨.)

* 주석 표현

주석 : 코드에 대한 설명 or 디버깅을 위해 작성되는 구문
1. HTML에서의 작성법 -> <!-- 주석내용 -->
2. CSS에서의 작성법 -> /* 주석 내용 */

* 엔티티

HTML 예약어를 기존에 사용하던 의미 그대로 사용하기 위해 별도로 만든 문자셋
(HTML 예약어를 HTML코드에서 그대로 사용하면 웹브라우저는 평소와 다르게 해석!)

* 엔티티 형태
-> &엔티티이름; or &#엔티티숫자(16진수);
