HTML이란

"Hyper Text Mark-up Language" 의 약자. 웹 페이지의 모습을 기술하기 위한 규약. 프로그래밍 언어가 아니라 마크업 언어다.

HTML 태그

시작&종료태그가 한 쌍으로 이루어져 있으며, 시작 태그만 있는 태그를 empty tag라고 한다.

HTML 요소

HTML 요소(element)는 여러 속성을 가질 수 있으며, 이러한 속성(attribute)은 요소에 대한 추가적인 정보를 제공. 요소는 <시작태그> ~ </종료>

ex> <p class="para"> 내용 </p>
